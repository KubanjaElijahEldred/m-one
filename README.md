# M-ONE Entertainment

Premium white-theme landing page for **M-ONE Entertainment** — Sound, Stage, Lights & Screen services in Uganda.

## Features

- 3-second splash screen with logo and loading bar
- Skeleton layout while images preload (no blank screen)
- Light white design matching the brand mockup
- Hero collage with floating event images
- Services, Gallery, Stats, and Contact CTA sections

## Stack

- React + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Production Build

```bash
npm run build
npm run preview
```
